# DistillPanel 语料条数重构

> 日期：2026-06-14 | 版本：v2.0 | 完整性：42/42 PASS

## 改动概要

将统计卡片的"知识条数"改为"语料条数"，值从概念数+关系数改为用户导入的原始语料条数。

## 新增文件

| 文件 | 说明 |
|------|------|
| `src/api/corpusStore.ts` | 语料条目 localStorage 持久化存储 |

## 修改文件

| 文件 | 改动 |
|------|------|
| `src/components/DistillPanel.tsx` | 统计卡片重命名 + 语料列表区段 + 蒸馏完成时保存语料 |

## 关键变更

1. **统计卡片**："知识条数 ▾" → "语料条数 ▾"，显示 `corpusEntries.length`，点击滚动到语料列表
2. **语料列表**：新增 `📂 语料列表` 区段，显示域标签、文本预览（120字）、概念数/关系数、时间戳，支持删除
3. **蒸馏保存**：蒸馏完成时自动保存语料条目（包含完整文本、域、概念/关系数）到 localStorage
4. **持久化**：关闭浏览器后重开，语料列表数据仍在
