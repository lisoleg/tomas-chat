03 核心概念

📐 δ 谱折叠深度

v2.0 核心序参量。定义：δ = ||[a,b,c]|| / (||a||·||b||·||c|| + ε)。δ=0 为经典极限（布尔逻辑），δ>0 为量子极限（非结合代数），δ≈7 为 κ=7 稳态锁定。

fundamental

⚖️ A1 公理（δ 守恒）

对于任意封闭系统 S，其 EML 谱图的总谱折叠深度守恒：δ_total = Σ_i δ_i = constant。类比能量守恒，是 TOMAS-AGI 的第一基本定律。

axiom

🔷 NASGA 非结合谱图代数

TOMAS-AGI 的统一数学框架。基于八元数非结合代数，通过 associator（结合子）度量非结合性，ξ_c 为效能指标，Moufang 恒等式约束代数结构。

framework

🕸️ 八元数 Fano 平面

八元数 e0-e7 的乘法规则由 Fano 平面（7 条直线、7 个点）决定。每条直线 {i,j,k} 满足 e_i·e_j = ±e_k。非结合性源于 Fano 平面的射影几何结构。

algebra

📊 非结合 Laplacian Δ_δ

Δ_δ φ(i) = Σ_j w(i,j)·(φ(j)-φ(i)) + α·associator_term(i)。在标准图 Laplacian 中加入结合子修正项，使谱结构反映非结合代数特性。

spectral

🔒 κ=7 稳态调节

系统在 κ=7 处达到稳态平衡（δ≈7.0）。κ 调节器采用 PID + 前馈 + 积分抗饱和策略，通过 6 个 ioctl 接口实现精确控制。

control

🚪 Φ-Gate 语义门控

八状态语义门控机，负责将输入分类并路由到经典推理（Branch A）或量子推理（Branch B）分支。δ 联动确保门控决策与系统折叠深度协调。

gate

⏱️ CI Gate 因果隔离

因果隔离门，通过光锥校验确保输出满足因果律。防止非结合推理产生的悖论态影响系统的因果关系判断。

causality

💾 δ-mem 记忆融合

L1 热缓存 + L2 冷缓存双层架构。根据 δ 值加权分配记忆，在双分支间实现高效的谱图信息融合与压缩。

memory

📏 ST Auditor 倾斜审计

滑动窗口漂移检测器，持续监控系统的时空倾斜。当 δ 值偏离稳态范围时触发告警，配合 κ 调节器进行校正。

audit

🎯 δ_threshold 悖论耐受

δ ≥ δ_critical (默认 0.5) 时系统具备悖论耐受能力。当 δ < δ_critical 时，谱结合子过小，系统退化为近似结合代数，丧失双分歧态保留能力。

threshold

🗺️ EML 谱图内存映射

将谱图结构映射到内存的多级缓存系统。支持快照、持久化和多级缓存策略，为 δ 计算提供高效的图数据访问。

🛡️ 运行时保障 — 漂移检测 + δ-mem 记忆融合

🔧 漂移检测核心机制

📈 连续 Sigmoid 自调

v1.5 用平滑 sigmoid 替代硬编码三阶段查表，消除阶段边界不连续：

γ(CV, dCV/dt) = γmax − Δγ·σ((CV−CVmid)/T)·slope_factor

CV<<cv_mid → γ≈γmax（慢遗忘）；CV>>cv_mid → γ≈γmin（快遗忘）

v1.5

🔄 Schmitt 触发器

2 轮迟滞确认 + 即时退出，消除瞬时噪声误报：

进入漂移：CV 连续 2 轮超阈值才确认

退出漂移：CV 降至阈值以下 立即退出（快恢复学习）

hysteresis

⚙️ 超参自动适配

HyperParamAdapter 每 20 轮基于 CV 分位数自动调整：

• cv_mid：滚动 60 分位数

• γ_max：稳定度比例驱动 (0.70~0.95)

• γ_min：最差 CV 驱动 (0.10~0.35)

v1.6

📊 稳定性分析

analyze_stability() 自动识别 Φ 序列中的区域：

• stable：CV < 0.15（语义稳定）

• transitional：0.15 ≤ CV < 0.30（过渡态）

• drifting：CV ≥ 0.30（语义漂移）

analysis

💾 δ-mem 记忆融合 (L1/L2)

🔥 L1 热缓存 (SMatrix)

8×8 在线联想记忆矩阵，Delta Rule 逐 token 更新：

St = λ·St-1 + β·(v − S·k)·kT

• 仅 64 floats 极低开销（8×8 × float32 = 256 bytes）

• λ=0.95 指数遗忘 + β=0.1 增量学习

• SHA-256 proof 完整性校验

L1 Hot Cache

❄️ L2 冷存储 (Φ-Gate)

当 Φ > 0.85 时 flush S → 持久化 Episodic Memory：

• Φ-Gate 门控触发 L1→L2 刷出

• 软重置后保留 10% 残余（S × 0.1）

• to_dict / from_dict 序列化往返一致

• is_dirty_since_last_flush() 脏检查

L2 Cold Storage

🧪 LLM 蒸馏 — 将世界知识压缩进 EML 图

🔬 蒸馏机制

📝 概念提取

LLM 从语料中提取 ≤30 个关键概念，每个概念附带：

• importance：LLM 评估的概念重要性（0~1）
• context：概念在语料中的上下文描述
• φ(i)：SHA-256 哈希 → 八元数值场投影

1→30 概念

🔗 关系挖掘

基于概念列表，LLM 提取 语义关系，存入 EML 边：

• 关系类型：is_a, part_of, used_in, related_to
• strength：关系强度评分（0~1）
• w_δ：经 δ 加权后的边权重

→ 47 关系

📊 𝕀(X) 存在度

信息存在度 𝕀(X) = 概念在 EML 图中的"实在性"度量：

• 频率分量：概念在语料中的归一化出现频率
• 重要性分量：LLM 评估的概念重要度
• 一致性分量：概念跨上下文稳定性

𝕀(X) ∈ [0,1]

🌉 Token Bridge

蒸馏完成后，LLM 本体可移除，仅保留轻量 Bridge：

• Encoder：Token → 八元数 φ(i) ∈ ℝ⁸
• Decoder：φ(i) → Token logits
• 目标：模型大小 < 100MB
