## 测试步骤

### 步骤 1：打开 DeepSeek Chat

1. 在浏览器中打开 **http://localhost:5173**
2. 看到欢迎屏幕，有两个模式卡片：
   - **💬 聊天模式**（默认）
   - **🔬 蒸馏模式**

### 步骤 2：切换到蒸馏模式

1. 点击 **🔬 蒸馏模式** 卡片
2. 或者点击左侧边栏的 **蒸馏图标** 按钮

### 步骤 3：加载 EML 文件（Token Bridge）

1. 滚动到 **Token Bridge** 区块
2. 点击 **"选择 .eml 文件"** 按钮
3. 选择 `tomas_agi/data/quantum_distilled_v2.eml`
4. 如果概念名称文件存在（`.concepts.json`），会自动加载
5. 加载成功后，会显示：
   - ✅ EML 图已加载
   - 顶点数：30
   - 边数：32

### 步骤 4：测试概念搜索

1. 在 **"概念搜索"** 输入框中输入：`量子计算`
2. 点击 **🔍 搜索** 按钮
3. 等待 2-3 秒，会显示：
   - **匹配置信度**：例如 92.3%
   - **匹配概念列表**：
     - 量子计算（相似度 0.923）
     - 量子比特（相似度 0.856）
     - ...
   - **子图大小**：扩展后的概念数 + 关系数

### 步骤 5：测试生成回复（无 LLM）

1. 确保概念搜索已完成
2. 点击 **💬 生成回复** 按钮
3. 等待 1-2 秒，会显示：
   - **生成回复** 区块
   - 标题：**无 LLM · 基于 EML 知识图谱**
   - 内容示例：
     ```
     关于「量子计算」，我找到了以下相关知识：

     【核心概念】
       1. 量子计算 — 信息存在度 δ=0.970，与「量子比特」等相关
       2. 量子比特 — 信息存在度 δ=0.950，与「量子门」等相关
       ...

      【关系网络】
        • 量子计算 相关于 量子比特
        • 量子比特 相关于 量子门
        ...

      【扩展知识】
        • 量子纠缠（δ=0.920）
        • 超导量子比特（δ=0.890）
        ...

     以上是基于已蒸馏知识库对「量子计算」的回答。
     ```

### 步骤 6：测试图谱可视化

1. 点击 **🕸️ 图谱可视化** Tab（在推理测试旁边）
2. 等待 2-3 秒，会显示 D3.js 力导向图：
   - **节点**：概念（大小 ∝ δ，颜色 ∝ 𝕀(X)）
   - **边**：关系（粗细 ∝ weight）
   - **交互**：
     - 鼠标滚轮：缩放
     - 拖拽节点：移动
     - 点击节点：查看详情（概念名称 + δ）
     - 悬停节点：高亮相邻节点

## 故障排除

### 问题 1：Dev server 无法启动

**症状**：`npm run dev` 报错

**解决方案**：
```bash
cd deepseek-chat
rm -rf node_modules
npm install
npm run dev
```

### 问题 2：EML 文件加载失败

**症状**：点击"选择 .eml 文件"无反应

**可能原因**：
- 文件格式不正确（不是有效的 EML 文件）
- 文件太大（> 10MB）

**解决方案**：
- 用 `token_bridge.py --info` 验证 EML 文件格式
- 如果文件太大，先蒸馏小语料测试

### 问题 3：概念搜索无结果

**症状**：搜索后显示"未能找到相关概念"

**可能原因**：
- EML 文件中的概念名称都是默认名称（`concept_0`, `concept_1`, ...）
- 没有加载概念名称伴侣文件（`.concepts.json`）

**解决方案**：
- 确保蒸馏时生成了 `.concepts.json` 文件
- 在加载 EML 文件时，同时加载 `.concepts.json` 文件
