### AEGIS 四阶段流水线（简化版）

```python
# extend_hypergraph.py（升级后）
class AEGISEngine:
    def evolve(self, trajectory, current_harness):
        # 1. Digester
        failed_edges = self.digester(trajectory)
        # 2. Planner
        proposals = self.planner(failed_edges, current_harness)
        # 3. Evolver
        candidate = self.evolver(current_harness, proposals)
        # 4. Critic+Gate
        accept, reason = self.critic_gate(candidate, trajectory)
        return (candidate, reason) if accept else (None, reason)
```
